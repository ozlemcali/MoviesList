# MoviesList
Movies List with OMDbAPI


## About Project
I made an application that searches for a movie using OMDB API and shows the properties of the movie.

## Includes

* SwiftUI
* Observable Object
* State Object
* MVVM Design Pattern
* SDWebImage
* SDWebImageSwiftUI
